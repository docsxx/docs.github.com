Module: Embedded

Read (and perhaps later write) embedded metadata from binary files and make a new embedded object (of class MiniExiftool) available so that it can be mapped to attributes of our model requires mini_exiftool gem which in turn requires the exiftool command line utility (http://www.sno.phy.queensu.ca/~phil/exiftool/index.html) written by Phil Harvey we also rely on attachment_fu attributes to access the binary file and attachment_fu’s workflow on uploaded files
