Module: Shippinglogic::Fedex::Rate

An interface to the rate services provided by Fedex. See the rate method for more info. Alternatively, you can read up on the rate services documentation on the FedEx website. It’s a great read.

rate

public rate(options = {})

This method can return all of the available services with their rates, transit times, etc. It can also return information on a specific fedex service.

Options

  • :shipper - a hash of address information.
    • :street_lines - street part of the address, separate multiple streets with a new line, dont include blank lines.
    • :city - city part of the address.
    • :state_or_province_code - state part of the address, use state abreviations.
    • :postal_code - post code part of the address, zip for the US.
    • :country_code - country code part of the address, use abbreviations, ex: ‘US’
    • :residential - a boolean value representing if the address is redential or not (default: nil)
  • :recipient - a hash of address information.
    • :street_lines - street part of the address, separate multiple streets with a new line, dont include blank lines.
    • :city - city part of the address.
    • :state_or_province_code - state part of the address, use state abreviations.
    • :postal_code - post code part of the address, zip for the US.
    • :country_code - country code part of the address, use abbreviations, ex: ‘US’
    • :residential - a boolean value representing if the address is redential or not (default: nil)
  • :service_type - one of SERVICE_TYPES, this is optional, leave this blank if you want a list of all available services. (default: nil)
  • :packaging_type - one of PACKAGE_TYPES. (default: YOUR_PACKAGING)
  • :packages - an array of packages included in the shipment. This is optional and should default to whatever your packing type default is in your FedEx account. I am also fairly confident this is only required when using YOUR_PACKAGING as your packaging type. This should be an array of hashes with the following structure:
    • :weight - a hash of details about a single package weight
      • :units - either LB or KG. (default: LB)
      • :value - the weight
    • :dimensions - a hash of details about a single package dimensions
      • :units - either IN or CM. (default: IN)
      • :length - the length
      • :width - the width
      • :height - the height
  • :ship_time - a Time object representing when you want to ship the package. (default: Time.now)
  • :dropoff_type - one of DROP_OFF_TYPES. (default: REGULAR_PICKUP)
  • :include_transit_times - whether or not to include estimated transit times. (default: true)
  • :delivery_deadline - whether or not to include estimated transit times. (default: true)
  • :special_services_requested - any exceptions or special services Fedex needs to be aware of, this should be one or more of SPECIAL_SERVICES. (default: nil)
  • :currency_type - the type of currency. (default: nil, because FedEx will default to your account preferences)
  • :rate_request_types - one or more of RATE_REQUEST_TYPES. (default: ACCOUNT)
  • :insured_value - the value you want to insure, if any. (default: nil)
  • :payment_type - one of PAYMENT_TYPES. (default: SENDER)
  • :payor - this is optional, if the person paying for the shipment is different than the account you specify then you can specify that information here. This should be a hash with the following elements. (default: nil)
    • :account_number - the Fedex account number of the company paying for this shipment.
    • :country_code - the country code for the account number. Ex: ‘US’

Simple Example

Here is a very simple example. Mix and match the options above to get more accurate rates: 

  fedex = Shippinglogic::Fedex.new(key, password, account, meter)
  fedex.available_services(
    :shipper => {
      :postal_code => "10007",
      :country_code => "US"
    },
    :recipient => {
      :postal_code => "75201",
      :country_code => "US"
    },
    :packages => [
      {
        :weight => {:value => 24},
        :dimensions => {:length => 12, :width => 12, :height => 12}
      }
    ]
  )

def rate(options = {})
  options = rate_defaults.deep_merge(options)
  normalize_rate_options(options)
  xml = build_rate_request(options)
  response = request(xml)
  parse_rate_response(response)
end
