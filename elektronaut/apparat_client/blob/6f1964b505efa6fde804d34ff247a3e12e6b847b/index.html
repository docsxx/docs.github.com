elektronaut / apparat_client

Apparat

Ruby SOAP client for the Apparat SMS service (http://www.apparat.no).

Installation

Install from GitHub: 

 gem sources -a http://gems.github.com
 sudo gem install elektronaut-apparat_client

Usage

 mc = Apparat::MessagingClient.new('my_private_key', :sender => 'elektronaut', :product_id => 1)
 mc.send(my_phone_number, 'Hello world!')
