README ==

Data Structures and Algorithms (DSA) is written in C#. The project contains data structures that are not in the .NET 3.5 framework as well as some which we (the DSA team) feel are better than those presented in the base class library (BSL) within .NET. The algorithms within DSA span several domains from strings, sorting, etc to maths.

The project homepage is http://codeplex.com/dsa. Please check here though for souce code.

Installation =

In order to build DSA you need only download the latest check-in to the remote git repository. We strongly advise that you build and then run ALL unit tests to verify behaviour. If all tests pass (which they should!) then you should make both a debug and release build for each respective scenario.

Structure =

All the data structures within DSA are located in the namespace Dsa.DataStructures. Similarly all algorithms are located in the Dsa.Algorithms namespace. The types in the Dsa.Utility namespace provide things like enforcing preconditions a runtime etc.
