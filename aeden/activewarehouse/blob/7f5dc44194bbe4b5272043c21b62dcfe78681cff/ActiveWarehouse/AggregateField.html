Class: ActiveWarehouse::AggregateField

  • ActiveWarehouse::Field
    • ActiveWarehouse::AggregateField

Encapsulates a fact column in a fact table. These fields represent columns that should be aggregated.

Instance Attributes

strategy_name [RW] public

Returns the value of attribute strategy_name.

Constructor

public initialize(fact_class, column_definition, strategy_name = :sum, field_options = )

fact_class is the class of the fact table this field is found in. column_definition is the ActiveRecord ColumnDefinition instance for this column. strategy_name is the name of th aggregation strategy to be used, defaults to :sum field_options is a hash of raw options from the original aggregate definition.

def initialize(fact_class, column_definition, strategy_name = :sum, field_options = ;{})
  super(fact_class, column_definition.name, column_definition.type, field_options)
  @column_definition = column_definition
  @limit = column_definition.limit
  @scale = column_definition.scale
  @precision = column_definition.precision
  @strategy_name = strategy_name
end
