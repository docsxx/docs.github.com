Module: OpenWFE::RepresentationMixin

This module gets included into the result of the expression pool (and engine) process_stack method (if the optional parameter ‘unapplied’ is set to true).

It adds a representation() method that returns an up-to-date representation of the executing process instance. This representation is suitable for ‘fluo’ rendering.

Public Instance Method Summary

#find_expression(fei)

Returns an expression given its FlowExpressionId.

#find_root_expression

Returns the root expression (the one with no parent expression) among the expressions in self (Array of FlowExpression instances).

#initial_tree

Returns the tree as it was when the process instance was launched.

#tree

Computes and returns the up-to-date representation of the process definition (on the fly changes included).

def find_root_expression

  self.find do |fexp|
    fexp.fei.expid == '0' &&
    ( ! fexp.is_a?(OpenWFE::Environment)) &&
    fexp.fei.is_in_parent_process?
  end
end
