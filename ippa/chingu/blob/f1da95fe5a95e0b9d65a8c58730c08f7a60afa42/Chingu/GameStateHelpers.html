Module: Chingu::GameStateHelpers

push_game_state accepts either a class inherited from GameState or an object-instance from such a class.

push_game_state(Intro): game state mananger will create a new Intro-object first time called and cache it.

push_game_state(Intro.new): The first line ends up calling "new" to Intro before activating the newly created game state. Each time ‘push_game_state(Intro.new)’ is called a new Intro-object will be created. Usefull for stuff like: push_game_state(Level.new(:level_nr => 11))
