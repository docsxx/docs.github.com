public execute(args)

A helper method to make your fire-and-forget requests easier

Parameters: args should be a Hash and is required 

  This Hash should at least contain <tt>:url</tt> and <tt>:method</tt> keys.
  You may also provide the following optional keys:
    <tt>:headers</tt> - should be a Hash of name/value pairs
    <tt>:response_header_handler</tt> - can be a string or object that responds to #call
      If an object was passed, it's #call method will be called and passed the current chunk of data
    <tt>:response_body_handler</tt> - can be a string or object that responds to #call
      If an object was passed, it's #call method will be called and passed the current chunk of data
    <tt>:payload</tt> - If <tt>:method</tt> is either <tt>:post</tt> or <tt>:put</tt> this will be used as the request body
