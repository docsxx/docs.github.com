haruska / ninja-decorators

ninja_decorators

http://github.com/haruska/ninja-decorators

DESCRIPTION:

Implements before_filter, after_filter, and around_filter decorators similar to Rails but can be used anywhere

FEATURES/PROBLEMS:

Currently, only around_filter is implemented. Also, need to change syntax to :only and :except filters similar to Rails

SYNOPSIS:

Similar to Rails, the following will execute the method common_around around both the foo and bar methods:

require ‘rubygems’ require ‘ninja_decorators’

class NinjaClass 

  include NinjaDecorators

  attr_accessor :ret

  around_filter :common_around, [:foo, :bar]

  def foo
    @ret <tt>= "foo"
  end

  def bar
    @ret </tt>= "bar"
  end

  private

  def common_around
    @ret = "common "
    yield
    @ret += " around"
  end

end

produces

irb(main):076:0> n = NinjaClass.new

> #

irb(main):077:0> n.bar

> "common bar around"

irb(main):078:0> n.foo

> "common foo around"

REQUIREMENTS:

rubygems activesupport >= 2.0.2

INSTALL:

sudo gem install haruska-ninja-decorators
