mongodb / activerecord-mongo-adapter

Class: ActiveRecord::ConnectionAdapters::MongoPseudoConnection

  • Object
    • ActiveRecord::ConnectionAdapters::MongoPseudoConnection

The database connection object used by ActiveRecord to talk to Mongo. Most of the actual communications with the database happens in the mongo_record modifications to ActiveRecord::Base, because it is that class (and its subclasses) that know what collection to talk to.

def method_missing(sym, *args)
  if $DEBUG
    $stderr.puts "#{sym}(#{args.inspect}) sent to conn"
    caller(0).each { |s| $stderr.puts s }
  end
end
