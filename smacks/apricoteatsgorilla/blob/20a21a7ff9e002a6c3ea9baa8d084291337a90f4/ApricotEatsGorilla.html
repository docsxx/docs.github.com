Class: ApricotEatsGorilla

ApricotEatsGorilla

Apricot eats Gorilla is a SOAP communication helper. It translates between SOAP messages (XML) and Ruby Hashes and comes with some additional helpers for working with SOAP services.

def [](source, root_node = nil)
  case source
  when String
    xml_to_hash(source, root_node)

  else
    nil
  end
end

def soap_envelope(namespaces = {})
  namespaces[:env] = "http://schemas.xmlsoap.org/soap/envelope/"

  xml_node("env:Envelope", namespaces) do
    xml_node("env:Body") { yield if block_given? }
  end
end

xml_to_hash

public xml_to_hash(xml, root_node = nil)

Translates a given xml String into a Ruby Hash.

Starts parsing at the XML root node by default. Accepts an optional root_node parameter for defining a custom root node to start parsing at using an XPath (Hpricot search).

The root node itself won’t be included in the Hash.

Examples

  xml = '<soap:Envelope xmlns:soap="http://schemas.xmlsoap.org/soap/envelope/">
      <soap:Body>
        <ns2:authenticateResponse xmlns:ns2="http://v1_0.ws.example.com/">
          <return>
            <apricot>
              <eats>Gorilla</eats>
            </apricot>
          </return>
        </ns2:authenticateResponse>
      </soap:Body>
    </soap:Envelope>'

  ApricotEatsGorilla.xml_to_hash(xml, "//return")
  # => { :apricot => { :eats => "Gorilla" } }

def xml_to_hash(xml, root_node = nil)
  doc = Hpricot.XML remove_whitespace(xml)
  root = root_node ? doc.search(root_node) : doc.root

  return nil if root.nil?
  return xml_node_to_hash(root) unless root.respond_to? :each

  return single_xml_root_node_to_hash(root) if root.size == 1
  multiple_xml_root_nodes_to_hash(root)
end
