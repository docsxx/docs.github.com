def initialize(source, root_node = nil)
  if source.kind_of? Hash
    initialize_from_hash source
  elsif source.respond_to? :body
    initialize_from_response source, root_node
  end
end

Dynamic Method Handling

This class handles dynamic methods through the method_missing method

public method_missing(method, *args)

Intercepts calls to missing methods and returns values from the response Hash in case the name of the missing method matches one of its key. Returns a new Savon::Response instance containing the value or returns the actual value in case it is not a Hash.

def method_missing(method, *args)
  value = value_from_hash(method)
  return value unless value.kind_of? Hash
  Savon::Response.new(value)
end
