Riwiki

Riwiki is Ruby on Rails plugin which adds wiki functionality to your application.

Installation

In your application directory call: 

  script/plugin install git://github.com/alno/riwiki

Usage

In your application directory call: 

  script/generate riwiki

It will generate:

  • WikiPageController to server wiki pages
  • WikiPage model to represent page
  • Migration to prepare database

Also it will add to your routes.rb something like: 

  map.riwiki_root '/wiki'

Template definition

You may create your own templates for controller actions (show, edit and history), in other case default built-in templates will be used.

Helper definition

Following helpers are defined by default and you may replace them with you own:

  • wiki_user - Renders user name or link by given user object. By default renders &lt;Unknown&gt; for nil and "User#{user.id}" for others.

Configuration

Configuration options are acessed via Riwiki.options object. Currently supported options:

  • user_class_name - Name of user model class. By default - ‘User’
  • formatter - Formatter instance, which process wiki content before output. It should have method format, which gets a string and returns it formatted. By default instance of Riwiki::Formatters::RedCloth is used (requires RedCloth gem). Other built-in formatter is Riwiki::Formatters::BlueCloth (requires BlueCloth gem).
