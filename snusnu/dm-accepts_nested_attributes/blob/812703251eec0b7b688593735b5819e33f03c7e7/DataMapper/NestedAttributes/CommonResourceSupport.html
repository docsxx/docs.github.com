snusnu / dm-accepts_nested_attributes

Module: DataMapper::NestedAttributes::CommonResourceSupport

This module provides basic support for accepting nested attributes, that every @see DataMapper::Resource must include. It includes methods that allow a resource to be marked for destruction and it provides an overwritten version of @see DataMapper::Resource#save_self that either destroys a resource if it’s @see marked_for_destruction? or performs an ordinary save by delegating to super
