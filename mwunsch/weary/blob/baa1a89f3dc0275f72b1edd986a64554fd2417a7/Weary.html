declare_resource

public declare_resource(name)

Define a resource.

Options that are allowed are:

:via
Get, Post, etc. Defaults to a GET request
:with
An array of parameters that will be passed to the body or query of the request.
:requires
Members of :with that are required by the resource.
:authenticates
Boolean value; does the resource require authentication?
:url
The url of the resource. You can use the same flags as #construct_url
:format
The format you would like to request.
:no_follow
Boolean; Set to true if you do not want to follow redirects.

def on_domain(domain)
  parse_domain = URI.extract(domain)
  raise ArgumentError, 'The domain must be a URL.' if parse_domain.empty?
  @domain = parse_domain[0]
end
