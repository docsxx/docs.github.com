A Todo (VTODO) calendar component groups properties describing a to-do Todos may have multiple occurrences

Todos may also contain one or more ALARM subcomponents to see the property accessing methods for this class see the RiCal::Properties::Todo module to see the methods for enumerating occurrences of recurring to-dos see the RiCal::OccurrenceEnumerator module

def method_missing(selector, *args, &b) #:nodoc:
  xprop_candidate = selector.to_s
  if (match = /^x_(.+)(=?)$/.match(xprop_candidate))
    if match[2] == "="
      add_x_property("x_#{match[1]}", *args)
    else
      x_properties[xprop_candidate]
    end
  else
    super
  end
end

def finish_time
  if due
    due_property.to_finish_time
  elsif duration_property && dtstart_property
    (dtstart_property + duration_property).to_finish_time
  else
    nil
  end
end
