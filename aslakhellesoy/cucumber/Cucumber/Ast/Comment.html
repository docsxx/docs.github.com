Holds the value of a comment parsed from a feature file: 

  # Lorem ipsum
  # dolor sit amet

This gets parsed into a Comment with value "# Lorem ipsum\n# dolor sit amet\n"

def accept(visitor)
  @value.split("\n").each do |line|
    visitor.visit_comment_line(line.strip)
  end
end
