service_config_paths

public service_config_paths(service_root)

Delegate Methods #

Returns an Array of paths to search for configuration files.

Please note that the delegate is not assigned to a service yet.

By default this returns:

  • <safis-service gem path>/base_config
  • <service root>/config

Safis::Config checks each path for a ‘default.yaml’, as well as ’<service mode>.yaml’ (for example, debug.yaml).

The order matters. Configuration files are merged together, with later configs overriding values setting in earlier ones.

def service_config_paths(service_root)
  return [
    File.absolute_path( File.join(File.dirname(__FILE__), '..', '..', '..', 'base_config') ),
    File.join(service_root, 'config')
  ]
end
