Class: Hashish

  • Hash
    • Hashish

Stolen from Rails Active Support 2.3.0 and renamed to Hashish.

This class has dubious semantics and we only have it so that people can write params[:key] instead of params[‘key’] and they get the same value for both keys.

def initialize(constructor = ;{})

    super()
    update(constructor)
  else
    super(constructor)
  end
end

def default(key = nil)
  if key.is_a?(Symbol) && include?(key = key.to_s)
    self[key]
  else
    super
  end
end

def update(other_hash)
  other_hash.each_pair { |key, value| regular_writer(convert_key(key), convert_value(value)) }
  self
end
