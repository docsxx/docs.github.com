Class: Micro::Node

Abstract node class. All nodes have potientally many outputs. Inputs are left to the concrete classes to define. Multiple outputs allow pattern nodes (and in the future branches) to be shared between trees

def state=(state)
  return if @state == state
  
  @state = state
  # notify input nodes that state has changed
  @outputs.each_value { |node| node.child_state_changed(@state) }
end
