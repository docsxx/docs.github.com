visfleet / micro_inferer

Micro Inferer

by VisFleet Labs

http://labs.visfleet.com/micro_inferer

Description

A micro inference engine.

Declare your rules: 

  inferer = Micro::Inferer.new
  inferer.when(:wet).it_is(:raining)
  inferer.when(:raining).and(:cold).it_is(:winter)

State the facts: 

  inferer.it_is(:wet)
  inferer.it_is(:cold)

Infer stuff with it: 

  inferer.infer
  > :winter

Why use an inference engine?

You may want to consider using a rules engine when:

  • Your rules have dependancies between them. In other words rule a is true only if b is true also
  • You have a large number of rules
  • The data your rules use changes infrequently
  • You need to run your rules frequently to see if they’ve fired
  • You want declarative (and pretty) syntax for expressing your rules

Performance

Performance is O© where c is the number facts that have changed. This is the main difference between a rules engine and a bunch of IF..THEN…ELSE statements. IF..THEN..ELSE statements have performance O(rc) where r is the number of rules.

Micro::Inferer’s performance is independant of the number of rules in the system

Install

  sudo gem install visfleet-micro_inferer
