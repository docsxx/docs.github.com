Class: Neo4j::Transaction::PlaceboTransaction

  • Neo4j::Transaction
    • Neo4j::Transaction::PlaceboTransaction

This is returned when trying to create a new transaction while a transaction is already running. This class will do nothing when the finish method is called. Finish will only be called when the ‘real’ transaction does it.
