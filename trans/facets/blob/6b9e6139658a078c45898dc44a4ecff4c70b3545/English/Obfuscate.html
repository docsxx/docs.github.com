Module: English::Obfuscate

String Obfuscation

The Obfuscate mixin provides a means of applying common scrambling patterns to strings.

This module is automatically mixed into the String class, but the mixin is designed in such a way that it can be mixed into any class that supports the #to_s method.

Examples 

  "superman".jumble  #=> "SUpeRmAn"

def self.included(base)
  if base.instance_methods.include?('replace') or
     base.instance_methods.include?(:replace) # Ruby 1.9
    base.module_eval{ include Replace }
  end
end

def dresner
  to_s.gsub(/\B\w+\B/){$&.split(//).sort_by{rand}}
end

def jumble
  j = ''
  to_s.split(//).each_with_index{ |c,i| j << ( i % 2 == 0 ? c.downcase : c.upcase ) }
  j
end
