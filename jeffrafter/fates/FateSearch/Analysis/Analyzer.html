Class: FateSearch::Analysis::Analyzer

Analyzers are used to break down text into suffixes. The base Analyzer is not intended for direct use and will not find any suffixes. Instead create a WhitespaceAnalyzer or SimpleIdentifierAnalyzer, or build a custom analyzer descendent that responds to append_suffixes
