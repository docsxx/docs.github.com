write

public write(html)

Writes the given html to the log window. Interpolating both Ruby(by use of #{ruby_variable}) and Javascript(by use of #[javascript_variable]) variables are supported.

Examples

 script << "var name = 'yawningman"
 Google::Log.write("Hello #[name], this is javascript interpolation.")

 name = "yanwingman"
 Google::Log.write("Hello #{name}, this is ruby interpolation.")
