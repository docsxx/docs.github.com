onload

public onload

Executes the content of the block upon the completion of page load. This uses Prototype’s Event.observe(window, ‘load’), or $(document).ready() if jrails is loaded.

Example: 

  # Generates:
  #   Event.observe("window", "load", function() { fn(); });
  # or:
  #   $(document).ready(function() { fn(); });
  page.onload do
    page.call 'fn'
   end

def onload
  if jquery?
    self << "$(document).ready(function() {\n"
  else
    self << "Event.observe(\"window\", \"load\", function() {\n"
  end
  yield
  self << "\n});"
end
