Class: Nanoc::Layout

A Nanoc::Layout represents a layout in a nanoc site. It has content, attributes (for determining which filter to use for laying out a page), a path (because layouts are organised hierarchically), and a modification time (to speed up compilation).

Instance Attributes

attributes [RW] public

A hash containing this layout’s attributes.

content [RW] public

The raw content of this layout.

mtime [RW] public

The time when this layout was last modified.

path [RW] public

This layout’s path, starting and ending with a slash.

site [RW] public

The Nanoc::Site this layout belongs to.

end
