Class: Liquid::Drop

A drop in liquid is a class which allows you to to export DOM like things to liquid Methods of drops are callable. The main use for liquid drops is the implement lazy loaded objects. If you would like to make data available to the web designers which you don’t want loaded unless needed then a drop is a great way to do that

Example:

class ProductDrop < Liquid::Drop 

  def top_sales
     Shop.current.products.find(:all, :order => 'sales', :limit => 10 )
  end

end

tmpl = Liquid::Template.parse( ’ {% for product in product.top_sales %} product.name } %endfor% ’ ) tmpl.render(‘product’ => ProductDrop.new ) # will invoke top_sales query.

Your drop can either implement the methods sans any parameters or implement the before_method(name) method which is a catch all

def invoke_drop(method)
  # for backward compatibility with Ruby 1.8
  methods = self.class.public_instance_methods.map { |m| m.to_s }
  if methods.include?(method.to_s)
    send(method.to_sym)
  else
    before_method(method)
  end
end
