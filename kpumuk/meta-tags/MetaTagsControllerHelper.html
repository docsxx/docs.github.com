Module: MetaTagsControllerHelper

Contains methods to use in controllers.

You can define several instance variables to set meta tags: 

  @page_title = 'Member Login'
  @page_description = 'Member login page.'
  @page_keywords = 'Site, Login, Members'

Also you can use set_meta_tags method, that have the same parameters as MetaTags.set_meta_tags.
